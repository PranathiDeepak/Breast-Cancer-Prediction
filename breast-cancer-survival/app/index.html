Project Overview

📊

Data Source

SEER registry data (2000–2017) from 17 geographic registries. Includes demographics, tumor biology, AJCC staging, receptor status, treatment variables, and survival outcomes.

🧮

Preprocessing

Median/mode imputation, one-hot encoding for Cox, label encoding for RSF & DeepSurv, StandardScaler normalization, and stratified 70/30 train-test split.

🌲

Random Survival Forest

200-tree ensemble with log-rank splitting. Handles nonlinear interactions and censored data without proportional hazards assumptions. Best C-index: 0.80.

🧠

DeepSurv Neural Net

Two hidden layers (128→64 neurons), ReLU activations, dropout regularization, Cox partial-likelihood loss, trained with Adam optimizer for 20 epochs.

Cox Proportional Hazards

The interpretable baseline model. Uses the partial likelihood formulation with ridge regularization (α = 1×10⁻⁴) to handle multicollinearity among AJCC staging variables.

DeepSurv Architecture

A feedforward neural network that learns a nonlinear risk function while preserving the Cox partial likelihood loss function. Dropout (p=0.2) prevents overfitting.

class DeepSurv(nn.Module): def __init__(self, input_dim=22): self.net = nn.Sequential( nn.Linear(input_dim, 128), nn.ReLU(), nn.Dropout(0.2), nn.Linear(128, 64), nn.ReLU(), nn.Dropout(0.2), nn.Linear(64, 1) # risk score ) # Loss: Cox negative log partial likelihood # Optimizer: Adam lr=1e-3, 20 epochs # C-index: 0.7900

Ensemble Risk Score

Normalized average of Cox and RSF risk predictions, scaled to [0, 1]. Combines the interpretability of Cox PH with the nonlinear power of RSF for more stable risk stratification.

# Normalize each model to [0, 1] cox_norm = (cox_risk - cox_min) / (cox_max - cox_min) rsf_norm = (rsf_risk - rsf_min) / (rsf_max - rsf_min) # Ensemble = average ensemble_risk = 0.5 * cox_norm + 0.5 * rsf_norm # Risk groups (terciles): Low / Medium / High # KM separation: monotonic across all follow-up times
